UltraTimer

Posted on 2018-08-31

As a new comer in swift coding, I started to build a timer app that can achieve basic functions of recording and displaying the time after a week’s learning. There are some existing relevant examples in github, but I couldn’t find one that can display multiple recorded time or use a table view to display unlimited time data. My timer, which I presumingly call it “UltraTimer”, reproduces the same function as the pre-installed timer in ios system. The realization of UltraTimer involves basic layout of UI components, application of UITableView, GCD Timer, and, most importantly, patience for debugging mysterious errors.

About NSTimer & GCD Timer

In most Github examples, NSTimer is used to keep the time; however, you may find the app stops keeping the time when you scroll the table. This is because the thread of UI controller conflicts with the thread where NSTimer belongs to. We can solve the problem by setting RunLoop of NSTimer, or, as what I did, use the GCD Timer, which functions in the main thread.
